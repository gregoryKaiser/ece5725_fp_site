Project Objective:

The objective of "It's Dangerous To Go Alone" is to demonstrate an interactive adventure game which requires one to obtain items through the detection of real objects using a Raspberry Pi camera. Additionally, we sought to demonstrate a remote connection for two Raspberry Pi's to communicate, allowing for a fun game to be played during social distancing in 2020. Additionally, we wanted to exercise our object-oriented programming to develop a starting point for future game level design and feature additions.
